# Payment Collection Platform

A Laravel-based payment collection platform designed for educational institutions and service providers to collect utility and billing payments through bKash.

## Project Highlights

- Integrated Payment Gateway
- Automated Transaction Verification
- Real-Time Payment Tracking
- Collection Reporting
- Reconciliation Management
- Audit Logging

## Payment Channels

- bKash
- Nagad
- Rocket
- Bank Transfer
- Card Payments
---

## Technology Stack

| Layer | Technology |
|---------|------------|
| Backend | PHP, Laravel |
| API Integration | cURL, REST API |
| Database | MariaDB |
| Server | Apache/Nginx |
| Version Control | Git, GitHub |

- Designed payment processing workflow

- Created reporting modules

## Business Impact

## Payment Flow

User → Laravel Application → Payment Gateway → Verification → MariaDB → Reporting Dashboard
